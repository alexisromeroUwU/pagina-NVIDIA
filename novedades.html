ANUNCIO DE NVIDIA OMNIVERSE CLOUD

NVIDIA ha anunciado hoy Omniverse Cloud, un conjunto de servicios en la nube que ofrece a artistas, creadores, diseñadores y desarrolladores acceso instantáneo a la plataforma NVIDIA Omniverse™ para la colaboración y simulación de diseño 3D desde hasta miles de millones de dispositivos. Entre los servicios de Omniverse Cloud se encuentra Nucleus Cloud, una sencilla herramienta de intercambio de "un clic para colaborar" que permite a los artistas acceder y editar grandes escenas 3D desde cualquier lugar, sin tener que transferir conjuntos de datos masivos. También incluye Omniverse Create, una aplicación para diseñadores técnicos, artistas y creadores para construir mundos 3D de forma interactiva en tiempo real; y View, una aplicación para que los usuarios no técnicos vean las escenas de Omniverse transmitiendo capacidades completas de simulación y renderizado utilizando la plataforma NVIDIA GeForce NOW™, impulsada por gpus NVIDIA RTX™ en la nube.
